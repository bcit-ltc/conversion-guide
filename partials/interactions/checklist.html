Feedback

Feedback to be revealed is indicated with the @ symbol.

Feedback should be part of the existing list. To add feedback, press Shift + Enter to create a new line within the list.

Checklists remember that items are marked as done unless you clear your browser's cache.

Developer Notes

Add the class strikethrough to the div if you want completed items to have a line through them.
